# JSonik
Task runner with JSON for monorepo, tasks | main.jsonik or main.jk

json + sonik = jsonik


## How to use

```shell
# run main.jsonik.json
jsonik run
jsonik run .
jsonik run main 
# run build.jsonik.json
jsonik run build
# run test.jsonik.json
jsonik run ./test.jsonik.json
# run {task_title}.jsonik.json
jsonik run {task_title}
```

## Build

```shell
jsonik build
```
